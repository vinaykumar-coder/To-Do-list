Car control using mobile using DTMF:
The commands are received by a DTMF decoder connected to an 8051 microcontroller, which then controls the vehicle movement with the help of a motor driver. Thus, with the motor driver IC, two DC motors enable vehicle movement based on commands supplied from the phone.
